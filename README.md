# ⚡ FractalTap

An interactive web application that lets you create beautiful lightning-like fractal patterns on a realistic wooden surface. A simple tap leaves a small burn mark, while pressing and holding generates an organically growing Lichtenberg-style fractal that continues to branch until you release.

## ✨ Features

- 🌳 Realistic full-screen wooden background
- 📱 Fully responsive on desktop and mobile devices
- 👆 Single tap/click creates a small burn mark
- ⚡ Tap-and-hold (or click-and-hold) grows a dynamic lightning-like fractal
- 🌿 Organic recursive branching with natural randomness
- 🎨 Smooth animations and immersive interaction
- 🖱️ Supports mouse, touch, and stylus input
- 🚀 Optimized for fluid performance across devices

## 🎮 Interaction

- **Single Tap / Click**
  - Creates a small burn mark at the selected location.

- **Tap & Hold / Click & Hold**
  - Begins growing a lightning-like fractal from the point of contact.
  - The fractal continuously expands and branches while the pointer is held down.
  - Releasing immediately stops the growth, leaving the completed pattern permanently etched into the wooden surface.
